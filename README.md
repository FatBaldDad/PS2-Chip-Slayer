# PS2-Chip-Slayer
A PlayStation 2 modchip enable/disable interface board for isolating reset and ground control, with documentation, test results, revision history, and installation notes.

## Availability

ChipSlayer hardware is currently not open-source.

This repository exists to document the project, provide testing data, installation information, compatibility notes, and revision tracking.

Production boards and kits are currently available only through official FBD Retro Game sales channels.
